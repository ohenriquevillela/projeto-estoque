# Sistema de Gestão de Estoque Full Stack

Este é um projeto de controle de estoque desenvolvido para colocar em prática conceitos de desenvolvimento Full Stack, integração de APIs e persistência de dados.

## Tecnologias Utilizadas

### Backend
- **Java 17 / Spring Boot**: Lógica de negócio e API REST.
- **Spring Data JPA**: Comunicação com o banco de dados.
- **PostgreSQL**: Banco de dados relacional.
- **Docker**: Containerização do banco de dados.

### Frontend
- **React + TypeScript**: Interface moderna e tipada.
- **Tailwind CSS**: Estilização rápida e responsiva.
- **Axios**: Consumo da API.
